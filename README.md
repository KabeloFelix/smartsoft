# SmartSoft

SmartSoft is a Python program that provides advanced file compression options on Windows. It allows users to reduce file sizes for storage and transfer by using different compression methods.

## Features

- **Multiple Compression Methods**: Choose between ZIP, BZ2, and XZ compression methods.
- **User-Friendly Interface**: Utilizes a simple GUI for file selection.
- **Efficient Storage**: Reduces file sizes efficiently for easy storage and transfer.

## Requirements

- Python 3.x
- Tkinter (usually included with Python)
- [Optional] Additional Python packages: `bz2`, `lzma`

## Installation

1. Clone the repository or download the `smartsoft.py` file.
2. Ensure you have Python 3.x installed on your system.

## Usage

1. Run the `smartsoft.py` script.
   ```bash
   python smartsoft.py
   ```
2. A file dialog will appear. Select the file you want to compress.
3. Choose the compression method:
   - `1` for ZIP
   - `2` for BZ2
   - `3` for XZ
4. The program will compress the file and save it in the same directory with an appropriate file extension.
